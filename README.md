# esp32-devkit-flasher
ESP serial flasher for esp32 devikit

## Hardware connection

Table below shows connection between two ESP32 devices.

| ESP32 (host) | ESP32 (slave) |
|:------------:|:-------------:|
|    IO26      |      IO0      |
|    IO25      |    RESET/EN   |
|    IO4       |      RX0      |
|    IO5       |      TX0      |
